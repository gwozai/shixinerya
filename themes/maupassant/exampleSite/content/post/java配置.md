## java jdk





JAVA_HOME、PATH、CLASSPATH(大小写无所谓)

变量设置参数如下：

- 变量名：**JAVA_HOME**
- 变量值：**C:\Program Files (x86)\Java\jdk1.8.0_91**     // 要根据自己的实际路径配置

- 变量名：**CLASSPATH**
- 变量值：**.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;**     //记得前面有个"."

- 变量名：**Path**
- 变量值：
  - **%JAVA_HOME%\bin;**
  - **%JAVA_HOME%\jre\bin;**

java、javac、java -verion  // 用这些命令来测试是否安装成功
