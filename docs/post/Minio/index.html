Minio

用docker安装minio

docker run -p 9000:9000 --name minio \
-d --restart=always \
-e "MINIO_ACCESS_KEY=#自己的账号" \
-e "MINIO_SECRET_KEY=#自己的密码" \
-v /home/data:/data \
-v /home/config:/root/.minio \
minio/minio server /data
