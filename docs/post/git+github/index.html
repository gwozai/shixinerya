git 一般的工作流程

  • 克隆git资源作为工作目录
  • 在克隆的资源上添加或修改文件
  • 如果其他人修改了，你可以更新资源
  • 在提交前查看修改
  • 提交修改
  • 在修改完成后，如果发现错误，可以撤回提交并再次修改并提交。

git add // 添加文件到暂存区
