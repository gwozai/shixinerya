在 mongodb 的容器当中

更新源 apt-get update

apt-get install vim

修改 mongo 配置文件

vim /etc/mongod.conf.orig

1.确保注释掉`# bindIp: 127.0.0.1` 或者改成`bindIp: 0.0.0.0` 即可开启远程连接
2.开启权限认证
security：
  authorization: enabled #注意缩进，参照其他的值来改，若是缩进不对可能导致后面服务不能重启

# 下载
curl -o https://fastd1.mongodb.org/linux/mongodb-linux-x86_64-3.0.6.tgz
# 解压缩
tar -zxvf mongodb-linux-x86_64-3.0.6.tgz
# 拷贝到指定目录
mv mongodb-linux-x86_64-3.0.6/ /usr/local/mongodb/data
# 创建数据存放目录与日志存放目录
mkdir -p /usr/local/mongodb/logs/ /usr/local/mongodb/
# 启动mongodb服务
/usr/local/mongodb/data --logpath=/usr/local/mongodb/logs/mongodb.log --port=27017 --fork
# 后期登录
/usr/local/mongodb/bin/mongo
